# Netflix Analytics Portfolio 

# Overview
This portfolio demonstrates **end-to-end analytics skills** by combining Python data cleaning with Tableau dashboards.  
It explores Netflix’s evolution in terms of catalog growth, content type, global reach, and ratings.

---

# Workflow
# 1. Python Preprocessing
- Cleaned raw Netflix dataset (`netflix_Raw_Data.csv`).
- Handled missing values, standardized release years, validated ratings.
- Exported cleaned dataset (`netflix_cleaned_data.csv`) for Tableau.

# 2. Tableau Visualization
- **Movies vs TV Shows** → Movies dominate Netflix’s catalog (~2x TV shows).  
- **Content Growth Timeline** → Rapid expansion during 2000–2020 streaming era.  
- **Worldwide Footprint Map** → US & India lead in content footprint.  
- **Growth vs Ratings** → Ratings dipped during peak expansion years.

---

# Key Insights
- Netflix transformed from a modest catalog to a global streaming giant.  
- Movies are more numerous and often higher‑rated than TV shows.  
- Smaller markets (e.g., Mauritius) differ sharply from global hubs.  
- Expansion years (2016–2019) saw a dip in average ratings.
